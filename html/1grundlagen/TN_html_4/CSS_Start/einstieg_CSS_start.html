Elemente mit CSS gestalten

Mit CSS wird das Aussehen vorhandener Elemente genauer bestimmt. Es gibt keine tags, aber Attribute. Hinter dem Attributnamen folgt ein : und danach ein Wert.
Damit der Interpreter die Umschaltung auf CSS versteht, gibt es mehrere Möglichkeiten. Eine davon ist, im head-Bereich des HTML-Dokuments ein style-Attribut einzufügen.

Die Grundfrage bei CSS-Angaben ist immer: wie sage ich dem Element, dass es gemeint ist?

Das Element muss irgendwie angesprochen werden, das macht man über Selektoren. Es gibt fünf Gruppen von Selektoren. Der einfachste davon ist der Element-Selektor, er bezieht sich direkt auf ein oder mehrere HTML-Elemente.

In diesem Beispiel sprechen wir den body an, die h1, figure und figcapture. Die Eigenschaften werden von einer geschweiften Klammer umschlossen. Mehrere Eigenschaften für ein Element werden mit einem Semikolon getrennt.
