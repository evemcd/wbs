semantische Tabelle

Mit dem <caption>Element können wir eine Tabellenüberschrift festlegen

wir können in HTML Tabellenbereiche semantisch strukturieren/definieren.
Dazu stehen uns die Elemente <thead><tbody><tfoot> zur Verfügung.
Die Elemente <thead> und <tfoot> dürfen innerhalb einer Tabelle jeweils nur einmal vorkommen. Das Element <tbody> kann durchaus mehrfach vorkommen.
